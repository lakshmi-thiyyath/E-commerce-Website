# JavaScript E-Commerce Project

A responsive e-commerce website built using HTML, CSS, and JavaScript as part of my frontend development learning journey.

## Features

* Product listings
* Shopping cart functionality
* Responsive design
* JavaScript-based interactions
* DOM manipulation

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Learning Goals

This project was built while following the SuperSimpleDev JavaScript course and helped me practice:

* JavaScript fundamentals
* Arrays and objects
* Functions
* DOM manipulation
* Responsive web design




## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open `amazon.html`  in a web browser.

Alternatively, you can use the VS Code Live Server extension:

1. Open the project in VS Code.
2. Right-click `amazon.html` file.
3. Select **Open with Live Server**.

The application will open in your browser.
